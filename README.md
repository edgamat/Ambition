# Project Ambition

This project is an example of using OpenTelemetry in a .NET application.

## Getting Started

To get started, you will need to have the following installed:

- .NET 8.0 SDK
- Docker

There are Docker Compose files in the `docker` folder to run:
- RabbitMQ
- SQL Server
- Seq
- Aspire Dashboard
